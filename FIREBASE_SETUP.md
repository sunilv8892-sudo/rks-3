# Content Editing Guide

CMS has been removed from this project.

## Current workflow
1. Edit markdown files directly in the content folder.
2. Commit and push your changes to GitHub.
3. Vercel deploys the latest version from the repository.

## Content paths
- Pages: content/home.md, content/about.md, content/facilities.md, content/terms.md, content/contact.md
- Committee: content/committee/*.md
- Gallery: content/gallery/*.md
- Uploads: images/uploads

## Important notes
- /admin is disabled and no longer provides editing tools.
- Netlify Identity and Git Gateway are not used.
